## Project Setup

```sh
git clone https://github.com/Aneal250/weather-app.git
```

### Clone the repo to your local machine

```sh
npm install
```

### Use npm install to install all the necessary dependencies.

```sh
npm run dev
```

### Run the app Locally using `npm run dev`

```sh
npm run build
```
